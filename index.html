Welcome to the GBT Pipeline Page.

In the coming year, GBT data will be made available to the user community through the NRAO data archive. In support of this release, and to offer quick access to GBT data, NRAO has instituted the GBT Pipeline Project to provide reduced spectra from public GBT datasets to the user community.

The GBT Pipeline Project has two goals.

The first is to produce an automated data processing pipeline that can generate quick-look spectra for about 80% of all spectral line data observed with the GBT, including data observed with the spectrometer and VEGAS in standard observing modes. For certain types of science data, this project will produce archive-ready data products, including a “summary sheet” for each reduced spectrum with accompanying header and statistical information.

A second goal of the Pipeline Project is to prepare tools that assist with certain potentially labor-intensive data processing tasks, such as calibrating and mapping KFPA observations.
